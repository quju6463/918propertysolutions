# 918 Property Solutions Publishing Checklist

## Best Simple Publishing Path

1. Create a GitHub repository for this folder.
2. Upload all site files to the repository.
3. Connect the repository to Cloudflare Pages, Netlify, or Vercel.
4. Buy or connect a custom domain.
5. Domain now selected: `https://918propertysolutions.co.nz`.
6. `sitemap.xml`, `robots.txt`, canonical links and social sharing image URLs have been updated for the domain.
7. Submit the site to Google Search Console.
8. Submit `https://918propertysolutions.co.nz/sitemap.xml` in Google Search Console.
9. Add Google Business Profile if the business will publicly serve clients.
10. Share the calculator pages on LinkedIn, Facebook property groups, email signatures, and client messages.

## Recommended Hosting

Cloudflare Pages is a good fit for this site because it is static HTML/CSS/JS, fast, low-cost, and easy to connect to a custom domain.

Netlify and Vercel are also suitable. The key is to publish from GitHub so every update can be redeployed cleanly.

## Weekly Content Updates

The local Codex automation named `Weekly market pulse update` is active and scheduled for Monday 9:00am NZ time.

Important: it updates the local workspace. It will not update the public website unless changes are pushed or deployed.

For a public site, choose one of these:

- Manual update: refresh figures locally, then upload or redeploy.
- GitHub workflow: update files, commit, push, and let Cloudflare Pages redeploy.
- Backend/API approach: build a proper data update service. This is more complex but best if you want live or scheduled data.

## Exposure Checklist

- Use clear page titles around NZ property calculators, flip calculators, and property due diligence.
- Keep each page focused on one purpose.
- Add useful internal links between Street Check, Flip Calculator, Profit Calculator and Contact.
- Use source notes where figures are examples or need verification.
- Avoid claiming live data unless the site actually pulls live data.
- Add new content over time: suburb checklists, flipping case studies, renovation budget examples, GST/tax explainers, and rental portfolio review guides.

## After Launch

- Test every calculator on mobile.
- Check all links.
- Submit sitemap to Google Search Console.
- Search Google for `site:918propertysolutions.co.nz` after a few days to confirm indexing.
- Share the most useful calculator page first, not just the homepage.
